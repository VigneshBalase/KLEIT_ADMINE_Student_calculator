Creating a new file in GitHub method

Follow these steps:

        
echo "# KLEIT_ADMINE_Student_calculator" >> README.md
git init
git add (file name){to add single file}/ 
git add . {for add all files in github}
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/VigneshBalase/KLEIT_ADMINE_Student_calculator.git
git push -u origin main
